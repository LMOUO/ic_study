# ic_study
本项目用于学习数字 IC 前端完整流程，包括：

1. project/doc/     相关说明文档
2. project/ip/      使用的硬核
3. project/libs/    使用的工艺库
4. project/rtl/     设计源码
5. project/sim/     设计仿真和验证仿真 脚本和生成文件
6. project/syn/     综合(Yosys) 和 静态时序分析(OpenSTA)
7. project/tb/      仿真v文件
8. project/Makefile 一键执行脚本文件

push.sh     项目管理推送git脚本
